Task Manager with AI Integration Documentation

Overview

The Task Manager is a Python-based GUI application for managing tasks with automatic tagging capabilities using Ollama AI.

Main Functions

load_tasks()

Loads tasks from the JSON file tasks.json. Returns an empty list if the file doesn't exist or is invalid.

Returns: list - A list of task dictionaries.
save_tasks(tasks)

Saves the list of tasks to the JSON file.

tasks (list): List of task dictionaries to save.

get_next_id(tasks)

Generates the next available task ID.

tasks (list): List of existing tasks.

Returns: int - The next available task ID.
add_task()

Adds a new task from the input field to the task list.

complete_task()

Marks the selected task as completed.

delete_task()

Deletes the selected task from the list.

update_task_list()

Updates the task listbox with current tasks and their status.

is_ollama_available()

Checks if Ollama is available on the system.

Returns: bool - True if Ollama is installed and accessible, False otherwise.
activate_ai()

Activates AI to automatically tag all tasks.

get_task_tags_from_ai(task_description)

Gets tags for a task using Ollama AI.

task_description (str): The task description to analyze.

Returns: list - List of tags for the task.
recommend_task_based_on_mood()

Recommends tasks based on mood and priority tags.

main()

Main function that initializes and runs the application window.

JSON File Structure

Tasks are stored in tasks.json with the following format:

[
    {
        "id": 2,
        "task": "rgrg",
        "completed": false,
        "tags": [
            "No Tags"
        ]
    }
]

Fields:

  • id: Unique task identifier
  • task: Task description
  • completed: Completion status (true/false)
  • tags: List of tags associated with the task

Usage Instructions

  1. Run the task_manager.py script to start the application.
  2. Use the input field to add new tasks.
  3. Select a task and use the "Complete" or "Delete" buttons to manage it.
  4. Use the "Activate AI" button to automatically tag tasks (requires Ollama installed).
  5. Use the "Recommend Task" button to get priority-based recommendations.

Requirements

  • Python 3.7
  • Modules: tkinter, json, os, subprocess
  • Ollama (optional, for AI features)
